# 📦 Acuamont - Sistema de Gestión y Ventas

Acuamont es un sistema web de **gestión y ventas** diseñado para optimizar el control de productos, clientes, pedidos y procesos administrativos.

El proyecto utiliza una arquitectura moderna full stack:

- ⚙️ Backend: Spring Boot  
- 🎨 Frontend: Angular  
- 🗄️ Base de datos: MySQL / PostgreSQL  

---

## 🚀 Características

- 📋 Gestión de productos  
- 👥 Administración de clientes  
- 🧾 Registro de ventas  
- 📦 Control de inventario  
- 🔐 Autenticación y autorización  
- 📊 Reportes básicos  
- 🌐 API REST  

---
